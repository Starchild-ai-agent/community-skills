# 风格参考：《我妈让我回家》

## 基本信息
- **类型：** 悬疑恐怖
- **时长：** 约10分钟
- **视角：** 第一人称（主观镜头）
- **镜头数：** 33个
- **结局类型：** 开放式反转

---

## 剧本结构参考

| 幕 | 名称 | 时长 | 核心情绪 |
|----|------|------|---------|
| 第一幕 | 普通的夜晚 | 1分30秒 | 正常，建立人物 |
| 第二幕 | 回家的路 | 3分钟 | 不对劲，异样累积 |
| 第三幕 | 到家 | 2分钟 | 压抑，进入危险区 |
| 第四幕 | 裂缝 | 2分钟 | 逻辑崩塌，恐惧爆发 |
| 第五幕 | 结局 | 40秒+字幕 | 反转，开放式恐惧 |

**恐惧递进节点：**
1. 导航方向反了（异样：技术故障）
2. 路口的老人（异样：人）
3. 门没有上锁（进入危险）
4. 妈妈说"我说了吗"（逻辑裂缝）
5. 手机里的妈妈VS沙发上的妈妈（崩塌）
6. 沙发空了（高潮）
7. "每次都有人接"（开放式延续恐惧）

---

## 分镜风格参考

### 手持晃动标准
- 正常状态：极轻微晃动（模拟呼吸）
- 紧张状态：明显手持晃动
- 恐惧爆发：剧烈抖动 + 轻微失焦

### 常用景别组合
- 手机/屏幕特写：信息传递节点
- 第一人称步行：移动过渡
- 驾驶视角固定：公路段落
- 凝视某物固定：压迫感制造

### 光线设计
- 第一幕：冷蓝屏幕光（孤独）
- 第二幕：昏黄路灯 → 越来越暗
- 第三幕：暖黄家居光（虚假安全感）
- 第四幕：暖光中透出冷意
- 第五幕：黑暗为主

## 爆款结构公式

```
熟悉感（建立安全区）
→ 第一个异样（小，可以解释）
→ 第二个异样（中，开始不安）
→ 第三个异样（大，无法解释）
→ 逻辑崩塌（不可能的事发生了）
→ 高潮反转（一个画面颠覆所有）
→ 开放式结局（恐惧在观众脑子里继续）
```
